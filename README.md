# Data Analysis and Visualization using Python

This project contains several data analysis tasks implemented using Python, Pandas, NumPy, and Matplotlib.

The goal is to practice data manipulation, numerical computing, and data visualization techniques commonly used in data science.

## Task 1 – Data Visualization using Pandas

A dictionary dataset was created and converted into a Pandas DataFrame.

Several visualizations were generated to analyze the data, including:

* Line plot
* Scatter plot
* Bar chart
* Pie chart

Multiple plots were combined in a single figure using subplot to provide a clearer comparison of patterns within the dataset.

## Task 2 – Comparison between Python List and NumPy Array

This analysis explores the differences between regular Python lists and NumPy arrays.

The experiment highlights:

* computational efficiency
* vectorized operations
* performance advantages of NumPy for numerical analysis

## Task 3 – Supermarket Sales Dataset Analysis

Using the **supermarket_sales dataset**, several analyses were performed:

* loading and exploring the dataset
* creating a new column `Total`
* analyzing sales performance by branch
* identifying the highest and lowest total sales
* creating multiple visualizations using Matplotlib

Visualizations include:

* bar chart of average sales per branch
* additional visual analysis to explore relationships between variables
